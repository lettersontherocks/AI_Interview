# AI Interview App

这是一个 iOS 面试练习应用程序，使用 SwiftUI 构建。

## 项目结构

- `AIInterviewApp/`: 主应用程序源代码
  - `App/`: 应用程序入口和主视图
  - `Models/`: 数据模型
  - `Services/`: 网络、音频、认证等服务
  - `Utils/`: 常量和扩展工具
  - `ViewModels/`: 视图模型
  - `Views/`: SwiftUI 视图组件

## 如何打开项目

1. 使用 Xcode 打开 `AIInterviewApp.xcodeproj` 文件
2. 选择适当的模拟器或连接设备
3. 点击 "Run" 按钮编译和运行应用

## 项目配置

- 最低部署目标：iOS 15.0
- 开发语言：Swift 5.0+
- 支持的设备：iPhone 和 iPad

## 权限要求

- 麦克风权限：用于录音面试回答
