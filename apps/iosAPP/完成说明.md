## 🚀 如何使用

### 1️⃣ 在Xcode中创建项目

```bash
# 打开Xcode
# File -> New -> Project
# 选择: iOS -> App
# 填写:
#   - Product Name: AIInterviewApp
#   - Interface: SwiftUI
#   - Language: Swift
#   - Minimum Deployment: iOS 15.0
```

### 2️⃣ 导入所有Swift文件

1. 在Xcode项目导航器中,右键点击项目根目录
2. 选择 "Add Files to AIInterviewApp..."
3. 选择 `AIInterviewApp` 文件夹
4. 勾选 "Copy items if needed"
5. 点击 "Add"

### 3️⃣ 配置Info.plist

复制 [Info.plist](Info.plist) 的内容到Xcode项目的Info.plist中,或者直接替换文件。

**重要权限**:
- ✅ 麦克风权限: `NSMicrophoneUsageDescription`
- ✅ HTTP访问: `NSAppTransportSecurity`

### 4️⃣ 配置后端地址

修改 [Constants.swift](AIInterviewApp/Utils/Constants.swift:3)中的`baseURL`:

```swift
struct Constants {
    static let baseURL = "http://47.93.141.137:8003"  // 修改为你的后端地址
}
```

### 5️⃣ 编译运行

```bash
# 在Xcode中
# 1. 选择目标设备(模拟器或真机)
# 2. 点击 ▶️ Run按钮
# 3. 或使用快捷键: Cmd + R
```

```
View (SwiftUI)
  ↓ 用户交互
ViewModel (@Published)
  ↓ 业务逻辑
Service (网络/音频/存储)
```

## 🐛 调试建议

### 常见问题排查

1. **编译错误**
   - 检查Xcode版本(需要14+)
   - 检查iOS版本(需要15.0+)
   - 清理缓存: Shift + Cmd + K

2. **网络请求失败**
   - 检查Info.plist中的HTTP权限
   - 检查Constants.swift中的baseURL
   - 确认后端服务正常运行

3. **音频录制失败**
   - 检查Info.plist中的麦克风权限描述
   - 确认在真机上测试
   - 检查设备权限设置

4. **界面显示异常**
   - 检查@Published属性绑定
   - 检查NavigationView嵌套
   - 使用Xcode的View Debugger
